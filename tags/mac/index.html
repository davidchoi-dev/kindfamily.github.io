13.1 모델 정의
테이블의 컬럼은 모델 클래스의 속성으로정의하고 테이블에는 메소드가 없지만 모델 클래스에는 메소드를 정의할 수 있습니다 이렇게 함으로써 테이블에 관련된 데이터와 행위를 모두 모델 클래스 한곳에서 정의할 수 있는 장점이 생깁니다.
